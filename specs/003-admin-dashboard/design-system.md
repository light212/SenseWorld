# SenseWorld 后台管理系统 - 设计系统

**项目**: SenseWorld Admin Dashboard
**版本**: v1.0
**日期**: 2026-03-27
**风格**: 参考 Vercel + Stripe，简洁专业

---

## 一、设计理念

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| **简洁** | 减少装饰，聚焦内容 |
| **专业** | 数据密集但不拥挤 |
| **清晰** | 层次分明，操作明确 |
| **高效** | 高频操作一键可达 |

### 1.2 视觉风格

**关键词**：简洁、专业、留白、微阴影、清晰层次

**参考**：
- **Vercel** - 大留白、微阴影、黑白为主
- **Stripe** - 专业、数据清晰、色彩克制
