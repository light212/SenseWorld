# SenseWorld 后台管理系统

## 概述

为 SenseWorld 多模态 AI 对话平台提供统一的后台管理能力，支持模型配置、用量监控、用户管理等功能。

## 一期功能（MVP）

### 1. 模型管理

#### LLM 配置
- 模型选择（支持多个 provider：OpenAI、阿里云、Longcat 等）
- API Key 管理（加密存储）
- 参数配置：temperature、max_tokens、top_p
- 默认模型设置

#### ASR 配置
- 模型选择（阿里云 Qwen ASR 等）
- API Key 管理
- 语言设置
- 采样率配置

#### TTS 配置
- 模型选择（qwen3-tts-instruct-flash 等）
- API Key 管理
- 音色选择（Cherry、其他）
- 语速、音量参数

#### VD 配置（预留）
- 视频理解模型配置
- 支持的格式设置

### 2. 用量监控

#### 调用统计
- 各模型调用次数（日/周/月）
- 各模型 token 消耗
- 各模型费用估算

#### 用户统计
- 活跃用户数
- 会话数
- 消息数

#### 告警配置
- 费用阈值告警
- 调用频率告警
- 错误率告警

### 3. 日志观测

#### 请求日志
- 每次 API 调用记录（时间、用户、模型、输入、输出）
- 延迟统计（ASR/LLM/TTS 各阶段耗时）
- 状态码、错误信息

#### 会话追踪
- 完整对话链路（用户输入 → ASR → LLM → TTS → 输出）
- 按会话 ID 查询
- 时间线可视化

#### 错误监控
- 错误聚合（按类型、模型、时间）
- 错误详情（堆栈、请求参数）
- 告警触发

#### 性能分析
- P50/P95/P99 延迟
- 吞吐量趋势
- 慢请求分析

### 4. 系统设置

#### 全局配置
- 默认模型组合
- 速率限制
- 超时配置

### 5. 终端管理

#### 终端类型
- Web 端（当前）
- iOS 端（后续）
- Android 端（后续）
- 小程序（后续）

#### 终端配置
- 各终端独立的模型配置
- 功能开关（语音/视频/文字）
- UI 参数（主题、布局）

#### 终端统计
- 按终端类型统计用量
- 各终端用户数、活跃度
- 终端版本分布

## 二期功能

### 6. 用户管理
- 用户列表 CRUD
- 角色权限
- 用量配额

### 5. Prompt 管理
- 系统 Prompt 模板
- 角色人设
- 场景配置

### 6. 知识库管理（RAG）
- 文档上传
- 向量化配置
- 检索测试

## 技术要求

### 后端
- 复用现有 NestJS/FastAPI 架构
- 新增 admin 模块
- 配置加密存储（API Key）

### 前端
- 独立的 /admin 路由
- 表格、表单、图表组件
- 权限控制（仅管理员访问）

### 数据库
- 新增表：model_configs, usage_logs, system_settings
- 配置热更新（不需重启服务）

## 优先级

| 功能 | 优先级 | 工作量 |
|------|--------|--------|
| 模型管理 | P0 | 3-5 天 |
| 用量监控 | P0 | 2-3 天 |
| 系统设置 | P1 | 1-2 天 |
| 用户管理 | P2 | 2-3 天 |
| Prompt 管理 | P2 | 2-3 天 |
| 知识库 | P3 | 5-7 天 |

## 核心用户场景

1. **运维人员**：快速切换模型，不需要改代码重启服务
2. **产品经理**：查看用量数据，分析用户行为
3. **管理员**：管理用户权限，控制成本
